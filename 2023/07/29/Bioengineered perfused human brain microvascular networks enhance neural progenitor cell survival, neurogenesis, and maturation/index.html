Programing Cell Assembly via Ink-Free, Label-Free Magneto-Archimedes Based Strategy

　　一种

Introduction

　　思路：

  • 神经退行性疾病导致人类认知能力下降，通常是由于神经元死亡或功能障碍所致。基于神经干细胞/神经祖细胞（neural stem cells and neural progenitor cells，NSCs/NPCs）的疗法被提议作为神经退行性疾病的治疗方法，因为这些细胞具有通过神经元修复和替代改善患者大脑功能的潜力。因此必须更好地了解影响NSC/NPC命运和行为的生物和物理机制。
  • 微血管能调节神经血管龛（neurovascular niche，NVN）中NSC/NPC的命运。内皮细胞是NSC/NPC自我更新和神经发生的协调者。在脑室下区（subventricular zone，SVZ）中，NPC（C型细胞）在进入神经发生或胶质细胞发生之前会在血管附近进行多轮细胞分裂，最终离开SVZ，沿着血管迁移到嗅球，在那里进一步分化为成熟的神经元。
  • 在过去十年中，人类NVN的体外模型得到了发展。然而，使用二维（2D）基质的培养物不能正确再现SVZ和颗粒下层（subgranular zone，SGZ）的细胞结构。使用三维水凝胶或细胞球的模型能更有效地复制NSCs/NPCs和内皮细胞之间复杂的细胞相互作用。然而，这些模型缺乏某些物理刺激，如血管内的管腔流和通过细胞外基质（ECM）的间质流（IF）。在大脑中，IF 可促进血管和神经细胞的相互作用、代谢物和生长因子的长距离运输，以及通过淋巴系统清除大分子废物。此外，中频的机械刺激还能调节细胞基因和蛋白质的表达，尤其是在微血管形成的过程中。一旦实现血管吻合，血管内血流对血管特性起着重要作用。除了输送氧气和营养物质外，血流还对血管壁产生剪切应力。这改变了内皮细胞基因表达和血管功能，包括调节血管龛中干细胞的行为。此外，内皮细胞的低通透性是血脑屏障（BBB）的一个重要特征，它将血液成分与脑组织区隔开来。在体外再现这些物理刺激和细胞结构对于开发生理上准确的人类NVN模型至关重要。
  • 由于微流控设备（MFDs）能对生化和物理变量进行时空控制，因此已成为模拟内皮细胞和非小体干细胞/全坏死细胞之间相互作用的流行系统。然而迄今为止还没有研究调查过间质液体的大量流动对NSC/NPC命运的影响。因此，目前还没有一种微流控模型能完全再现人体NVN的细胞结构和物理刺激。

NPCs were capable of self-renewal, neurogenesis, and astrogenesis in 2D cell culture

  • （A）NPC在二维细胞培养中的分化示意图。
    • 在星形胶质细胞分化方案中，星形胶质细胞分化培养基（ADM）用于将NPC转化为星形胶质细胞前体细胞（APreCs）。然后用星形胶质细胞成熟培养基（AMM）将APreCs转化为NPC衍生的星形胶质细胞（NPC-ACs）；

IF enhanced neurogenesis of NPreCs cocultured with BMVNs in MFDs

在这项工作中，我们证明了将 NPreC-Nsphs 与由 BECs-tdT、PCs 和 ACs 组成的 BMVNs 共同培养的益处。由于我们之前发表的论文详细说明了 IF 在体外发育 BMVNs 中的关键作用（30），因此之前讨论的所有涉及 NPreC-Nsphs 的结果都是在 MFDs 中进行的，并在水凝胶通道上施加了 IF。

为了强调 IF 在四联培养过程中对神经发生的有利影响，我们用 NPreC-Nsphs 重建了 NPreC-BEC-PC-AC 培养物，并在流动或静止条件下维持样本（图 7A）。1 周后，在流动和静态样本中都能观察到神经球周围广泛的 BMVNs（图 7B）。与静态样本（40.93 ± 5.10%；图 7C）相比，流动样本中的 BMVNs 的血管面积（67.30 ± 3.87%）明显增大。虽然静态培养中微血管分支平均直径的平均值（26.41 ± 2.99 μm）低于流动培养中的平均值（43.14 ± 5.57 μm），但两者之间并无显著差异（图 7D）。同样，静态条件下血管节段的平均数量（5.00 ± 1.20）大于流动条件下（1.00 ± 0.67），但未发现明显差异（图 7E）。虽然我们发现这些结果出乎意料，但考虑到我们之前发表的论文（30），我们将缺乏统计学意义归因于分析的样本数量较少。我们曾考虑为每个样本采集多个感兴趣区（ROI）以生成平均值，但很难找到没有 NPreC-Nsphs 的 BMVNs 区域，这将影响微血管分析。

尽管有微血管分析的结果，我们还是继续分析了 NPreC-Nsphs 的特征，以确定应用 IF 是否会影响四联培养中的神经发生。1 周后，观察到在流动和静态条件下培养的 NPreC-Nsphs 将 Tuj1+ 神经元延伸到纤维蛋白凝胶中（图 7，F 和 G）。然而，与流动条件下培养的神经球相比，静态培养的神经球明显更小，神经元网络的范围也更小。为了量化 IF 对 NPreC-Nsph 神经发生的影响，我们测量了在流动和静态条件下培养的神经球的神经球核面积、神经元生长总量和最大神经元延伸率。结果发现，与静态条件下的神经球（14,238 ± 2902 μm2；图 7H）相比，流动条件下的神经球平均核心面积（18,629 ± 2997 μm2）有所增加。同样，神经球的平均总神经元生长量在流动条件下（19.61 ± 0.48 mm）大于静态条件下（14.08 ± 0.62 mm；图 7I）。最后，在流动和静止条件下培养的神经球的平均最大神经元延伸率没有统计学差异，分别为 192.55 ± 7.66 和 201.90 ± 25.24 μm（图 7J）。不过，这也在意料之中，因为在两种培养物中观察到的 NPreC-Nsphs 的最长神经元通常长度相当。这些数据共同表明，在四联培养过程中应用 IF 有利于 NPreC-Nsph 的神经发生。我们推测，通过纤维蛋白凝胶的大量液体运动为神经球提供了充足的可溶性生长因子，使其能够进行神经元分化并延伸出健康的神经元。

观察到荧光微球流经在流动条件下形成的 BMVN（图 7K 和影片 S1）。没有对静态样本进行微球灌注试验，因为我们以前曾报道过，如果不使用 IF，BMVNs 无法实现吻合（30）。此外，在静态条件下培养的微血管在正常培养过程中不会出现任何管腔流动，因为水凝胶通道上没有静水压力梯度。在流动条件下培养的样本中，观察到 Tuj1+ NPreC-Nsphs 与灌注的微血管非常接近，表明它们在培养过程中得到了管腔流动的微血管的支持（图 7K 和影片 S1）。此外，我们还观察到 Tuj1+ 神经元和 tdTomato+ 微血管之间的直接接触（图 7L）。这些结果令人鼓舞，因为它们展示了体外模型中神经发生和血管生成的同步过程。结合在流动条件下观察到的增强的神经球核心面积和神经元总生长量，这些数据证明了在 NPreC-Nsphs 四重培养中实施 IF 的合理性。

我们证实了 IF 对于维持与 BMVNs 一起培养的 NPreC-Nsphs 在 MFDs 中的重要性，之后我们想确定将 NPreCs 培养成分散的细胞是否会改善神经发生。为了验证这一点，我们在分化培养基 EGM-2:NMM 下，在流动或静止条件下将 NPreCs-ZsG 与表达 tdTomato 的脑微血管内皮细胞（BMECs-tdT）、PCs 和 ACs 一起培养（图 8A）。用 ZsGreen（NPreCs-ZsG）标记 NPreCs，以便跟踪和量化总细胞群及其各自的命运。2 周后，在流动样本和静态样本中都观察到了 BMVNs（图 8，B 和 C）。在这两种情况下，都能观察到 NPreCs-ZsG 分散在微血管之间的间隙中（图 8，D 和 E）。在 ZsGreen1+ 和 ZsGreen1- 细胞中都观察到了 Sox2 表达，这分别确定了自我更新的 NPreCs-ZsG 和 ACs 群体中的少量 Sox2+ 细胞（图 8，B 和 D）。同样，在 ZsGreen1+ 和 ZsGreen1- 细胞中都观察到了 MAP2 表达，这分别确定了分化的 NPC 神经元和 ACs 群体中的 MAP2+ 细胞（图 8，C 和 E）。此前，我们观察到在二维细胞培养中，人胎儿 ACs 的某些细胞会随机表达一定量的 MAP2（图 S6A）和 Sox2（图 S6B），这给分析 Sox+ 数据带来了困难。不过，由于在 NPreCs-ZsG 中标记了 ZsGreen1，我们能够将属于 NPreCs-ZsG 的 Sox2 和 MAP2 信号与属于 ACs 的信号区分开来。由此，我们能够在流动和静止条件下，对四联培养 2 周后的 NPreC-ZsG 干性和神经发生进行表征。在流动条件下观察到的 ZsGreen1+ 细胞的平均总数（136.55 ± 6.53）明显高于静态条件下（86.91 ± 6.42），这表明应用 IF 2 周后出现的 NPreCs-ZsG 多于未应用 IF 的情况（图 8F）。在流动（41.80 ± 2.20）和静止（46.80 ± 3.20）条件下，Sox2+/ZsGreen1+细胞的平均数量无明显差异（图 8G）。鉴于流动样本中发现的 ZsGreen1+ 细胞总数较多，这导致流动条件下 Sox2+/ZsGreen1+ 细胞的百分比（28.32 ± 1.99%）低于静态条件下（47.89 ± 1.48%；图 8H）。

  • （F-J）第14天在流动和静态样本中测得的ZsGreen1+细胞数（F，n = 11）、Sox2+/ZsGreen1+细胞数（G）和百分比（H）（n = 5）以及MAP2+/ZsGreen1+细胞数（I）和百分比（J）（n = 6）的图表。这些数据表明，IF 增加了四联培养中 MAP2+/ZsGreen1+ 细胞的总数，但并没有增加表达 MAP2 的 NPreCs-ZsG 的百分比，这表明 IF 主要通过增加细胞增殖总数来促进神经发生，然后向神经元系分化，从而产生更多的神经元，而不是偏向分化的百分比。最结合血管生成实验的结果，可以比较肯定地说，NPreCs 对血管生成没有任何积极影响。虽然这些结果表明 NPreCs 不会影响内皮细胞的形态发生，但之前报道的所有 BECs 与分散的 NPreCs 的共培养都伴随有 PCs 和 ACs，以支持微血管网络的正常形成（图 8，B 和 C）。
